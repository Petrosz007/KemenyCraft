# Server Setup

Install required tools

    sudo apt update
    sudo apt install curl unzip zip
    curl -s "https://get.sdkman.io" | bash
    source "$HOME/.sdkman/bin/sdkman-init.sh"

Check if SDKMan has been installed correctly:

    sdk version

Install Java 17

    sdk install java 17.0.3-tem
